## 인공지능의 역사
- 준비기
- 요람기
- 첫 번째 겨울
- 발전기
- 두 번째 겨울
- 안정기
- 부흥기

## 딥러닝의 탄생 이유
- 알고리즘 : 역전파, 경사하강법 기반 깊은 신경망 모델
- GPU 기반 병렬 처리 : 딥러닝 개발에 필수적인 행렬 연산 시간 크게 단축
- 빅데이터 : 웹, 센서 등 다량의 자료가 학습에 활용

## 인공지능의 다양한 분야에의 도전
- 기사 작성
- 시와 소설 작성
- 작곡
- 그림 그리기

## 생성형 인공지능(Generative AI)
새로운 콘텐츠를 생성하는 인공지능 기술
- 텍스트 생성
- 그림 생성
- 영상 생성

먼저 입력된 트레이닝 데이터의 패턴과 구조를 학습하고, 유사 특징이 있는 새로운 데이터를 만들어 냄
