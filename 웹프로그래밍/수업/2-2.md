## JavaScript 기본 표기
문장(statement) 규칙
- 문장의 맨 끝에 semicolon(;)을 붙임
  - 없어도 에러는 발생하지 않지만 비추천
- 문장 도중에 공백이나 개행, 탭 포함 가능
- 대 / 소문자 구분

comments
- // comment : 단일행 주석
- /* comment */ : 복수행 주석
- /** comment */ : 문서화 주석

변수 선언
- var 명령어
  - 초기값이 없이 변수명만 선언한 경우, 변수에 undefined 값 할당
    - ex. var msg; var x = 10;
